# Launch Checklist

## One-time setup

- [ ] Use `ORCHESTRATE` as the comment trigger.
- [ ] Load the DM and email text from `DELIVERY-COPY.md` into the automation.
- [ ] Confirm the automation sends the giveaway URL, not the Team Assets ZIP.
- [ ] Apply the intended delivery tag.
- [ ] Route failed deliveries and support questions to a person.

## Before publishing

- [ ] Open the giveaway URL in a private browser window.
- [ ] Confirm the downloaded filename says `Giveaway`.
- [ ] Extract the ZIP and open `START-HERE.md`.
- [ ] Post the infographic with `SOCIAL-POST.md`.
- [ ] Submit one test comment using `ORCHESTRATE`.
- [ ] Confirm the message or email arrives with the correct link.

## After publishing

- [ ] Check the first real delivery end to end.
- [ ] Use `COMMENT-REPLIES.md` only for questions or failed automation.
- [ ] Do not manually assemble files or send repository source.
- [ ] Do not send the Team Assets ZIP to leads.
